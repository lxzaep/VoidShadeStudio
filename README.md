Professional shader installer for Roblox Studio powered by ReShade technology.

VoidShadeStudio provides a streamlined way to install and manage cinematic shader presets designed for Roblox Studio creators.

## Requirements

- Windows 10 or Windows 11 (64-bit)
- Roblox Studio
- NVIDIA or AMD GPU recommended

---

## Features

- One-click installation
- Automatic Roblox Studio detection
- Preset management
- Easy updates
- Lightweight installer

---

## Installation

1. Download the latest release.
2. Run the installer.
3. Follow the setup wizard.
4. Launch Roblox Studio.
